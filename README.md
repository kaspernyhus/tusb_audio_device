# TinyUSB Audio Device example
Change audio configuration in `menuconfig` ---> `Component config` ---> `ESP TinyUSB Audio` ---> `Audio Device Class (UAC2)`

> NOTE:
> For Audio functionality `Use Synopsis dwc2 driver` has to be selected


## Supports:
- 48kHz sample rate
- 16bit/24bit samples
- 1-4 channels
